A Salary Component History view that shows how salary components and their percentages have changed over time.

🖥️ Page Structure
1. Header Section (Inside a Container/Card)
Background: white or very light grey
Padding: medium–large
Content:
Title (large, bold):
“Salary Component History”
Subtitle (smaller, muted):
“Historical salary component configurations and percentages”
Divider line below header (subtle horizontal rule)
2. Table Container
Full-width inside the page/container
Background: white
Border: light grey
Rounded corners (optional)
Horizontal scroll enabled on smaller screens
📊 Table Structure
Columns (Left → Right)
Start Date
End Date
Basic Salary
BS Percent
Children Allowance
CA Percent
Complementary Allowance
COMA Percent
Other Allowance
🧱 Table Header Row
Background: light grey
Text: bold, dark
Alignment:
Center for most columns
Dates can be left or center aligned


Use a card-style container with a light background and subtle shadow
Table should have clear column headers with slightly bold text
Use green check icons (✓) for enabled items
Use a red cross icon (✗) for disabled items
Align percentages as text (e.g., “100%”)
Maintain consistent spacing and padding for readability
Use a modern, minimal dashboard design style
